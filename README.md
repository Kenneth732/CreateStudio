### Run Locally

To run **Portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Kenneth732/CreateStudio.git
```

Windows:

```bash
git clone https://github.com/Kenneth732/CreateStudio.git
```
